# Quick Snake
A fast paced terminal snake game written in C++17

![Fast paced to test your skills](https://github.com/gregstula/quick-snake/blob/master/images/qsnake.gif)

Requirments:
  - a UTF-8 Terminal
  - ncurses
  - clang

Intallation:

    git clone https://github.com/gregstula/quick-snake.git
    cd quick-snake
    make
    ./qsnake
